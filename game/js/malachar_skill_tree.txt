================================================================================
                              DESIGN NOTES
================================================================================

BALANCE PHILOSOPHY:
- Levels 1-25:  Normal power curve, establishing build identity
- Levels 26-50: Strong power spike, build comes together
- Levels 51-75: God-tier powers emerge, dominant in most content
- Levels 76-100: Reality-breaking endgame powers, tackle hardest content

REACHING HIGH LEVELS:
- Most runs end at level 20-30 (balanced, fun gameplay)
- Level 50+ requires good build synergy and skill
- Level 75+ requires near-perfect build and execution
- Level 100 should feel incredible but achievable with practice

CROSS-PATH SYNERGIES:
- Legion + Champion = Massive army of powerful minions
- Legion + Reaper = Army sustains Malachar, Malachar empowers army
- Champion + Reaper = Quality over quantity, Malachar + strong minions
- All Three Paths = Balanced, versatile, adaptable to any situation

MOBILE CONTROLS:
- Movement only (virtual joystick or tap-to-move)
- Everything triggers automatically based on conditions
- Build complexity comes from upgrade choices, not execution
- Perfect for multiplayer coordination

MULTIPLAYER CONSIDERATIONS:
- Different players can specialize in different paths
- Buffs and effects can benefit party members
- Shared XP helps new players catch up
- High-level players can carry but everyone contributes

ENDLESS PROGRESSION:
- No "win" state, world scales infinitely
- Level 100 is a milestone, not the end
- Each run feels different based on upgrade choices
- Always something to push for
